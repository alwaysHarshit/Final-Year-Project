# Client

This is the front-end of the project.  
It allows users to interact with the system, send data or requests to the master server, and view processed results.

### Tech Stack
- React (or React Native)
- Axios for API calls
- Styled with modern UI components

### Runs On
Local port: `http://localhost:5173` (or your chosen dev port)

### Commands
```bash
npm install
npm start
